# 事故情報スライドショー v0.2

現場タブレットで事故情報を自動でスライド表示するアプリ。

## ファイル構成

```
jiko_slideshow/
├── index.html      ← スライドショー本体
├── slides.json     ← 表示する事故情報の管理表
├── pdfs/           ← PDFファイルを入れるフォルダ
│   ├── 20260303-01.pdf
│   └── ...
└── README.md       ← これ
```

## 使い方

### ローカルで動作確認する場合

`index.html` をブラウザで直接開いてもPDFが表示されない。簡易サーバー経由で開く必要あり。
今回はVercelにデプロイして使う前提。

### Vercelにデプロイする場合

1. [Vercel](https://vercel.com) にログイン
2. 「Add New Project」→「Deploy」
3. この `jiko_slideshow` フォルダごとドラッグ&ドロップ
4. 発行されたURLをタブレットで開く

## 新しい事故情報を追加する手順

1. PDFを `pdfs/` フォルダに入れる（ファイル名は `YYYYMMDD-NN.pdf` 形式）
2. `slides.json` に1件追加する：
   ```json
   {
     "file": "20260501-01.pdf",
     "title": "事故のタイトル",
     "date": "2026-05-01",
     "severity": "軽傷",
     "comment": "",
     "visible": true,
     "durationSec": 30
   }
   ```
3. Vercelに再デプロイ（フォルダごとドラッグ&ドロップ）

## slides.jsonの項目説明

| 項目 | 意味 | 例 |
|---|---|---|
| file | PDFのファイル名 | `20260303-01.pdf` |
| title | 表示するタイトル | 立防下部の笹竹伐採中に…|
| date | 事故発生日（表示用）| `2026-03-03` |
| severity | 重大度 | `軽傷` / `重傷` / `死亡` / `物損` |
| comment | 副社長コメント（v0.2では未使用、将来用）| |
| visible | 表示する？ | `true` or `false` |
| durationSec | 表示秒数 | `30` |

## 使い捨てるとき

- 一時的に表示を消したい → `visible` を `false` にする
- 完全に削除したい → `slides.json` から該当行を消す＋`pdfs/` のファイルも消す

## 操作方法（スライドショー画面）

- **何もしなくても自動で次に進む**（デフォルト30秒）
- マウスを画面下に持っていくと操作バーが出る（前へ / 停止 / 次へ / 秒数変更）
- キーボード：`←` `→` で送り、`スペース` で停止/再開
